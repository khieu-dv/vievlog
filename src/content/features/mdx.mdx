# MDX

MDX allows you to use JSX in your Markdown content. You can import and use React components directly in your documentation.

## Basic Usage

```mdx
import { Callout } from 'nextra/components'

# My Page

<Callout type="info">
  This is a callout component!
</Callout>

Regular markdown content can be mixed with components.
```

## Components

You can use various built-in components:

### Callout

```mdx
<Callout type="info">
  Info callout
</Callout>

<Callout type="warning">
  Warning callout  
</Callout>

<Callout type="error">
  Error callout
</Callout>
```

### Code Blocks

Code blocks support syntax highlighting:

```javascript
function example() {
  return "This is highlighted JavaScript code"
}
```

```python
def example():
    return "This is highlighted Python code"
```
