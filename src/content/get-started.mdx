# Get Started

Learn how to get started with this documentation system.

## Installation

This project uses Nextra for documentation. The following dependencies are installed:

```bash
npm install nextra nextra-theme-docs
```

## File Structure

The documentation files are located in the `src/content/` directory:

```
src/
├── content/
│   ├── _meta.ts          # Navigation metadata
│   ├── index.mdx         # Homepage
│   ├── get-started.mdx   # This page
│   └── features/         # Features section
└── app/
    └── docs/
        └── [[...mdxPath]]/
            └── page.tsx  # Dynamic route handler
```

## Writing Documentation

You can write documentation using Markdown (`.mdx` files). Here's an example:

```markdown
# Page Title

This is a paragraph with **bold text** and *italic text*.

The `_meta.ts` files control the navigation structure and page titles.
