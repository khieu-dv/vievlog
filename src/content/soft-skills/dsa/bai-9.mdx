    // B·∫Øt ƒë·∫ßu t·ª´ ph·∫ßn t·ª≠ th·ª© hai (index 1)

        // T√¨m v·ªã tr√≠ ch√®n b·∫±ng binary search (t√πy ch·ªçn)
